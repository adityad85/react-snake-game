# React Snake Game
> Uses React and ES6 constructs to build a simple snake game.

Feed as much as you can to your snake without eating your own body.
As soon as you collide with yourself the game ends.

![](header.png)

## Installation

OS X & Linux:

```sh
npm install or yarn
```

Windows:

```sh
npm install
```

## Development setup

Just clone the repo and run below command to start the game.

```sh
yarn start
or
npm run start
```
